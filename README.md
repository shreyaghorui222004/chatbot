## рҹ©ә What is MediBot?

**MediBot** is a polite, conversational AI medical assistant built with **Streamlit**<br/>
and powered by **Groq's LLaMA 3.1 8B** model via **LangChain**.<br/>
It listens to your health concerns and offers calm, concise, practical advice вҖ”<br/>
like having a knowledgeable friend on call, anytime.

<br/>

> вҡ пёҸ MediBot is an informational assistant only. It does **not** replace professional medical advice.<br/>Always consult a licensed healthcare provider for serious concerns.

---

## вңЁ Features

| Feature | Description |
|:---:|:---|
| рҹ’¬ **Conversational Memory** | Full multi-turn chat вҖ” MediBot remembers your conversation context |
| рҹ§  **LLM-Powered** | Backed by `llama-3.1-8b-instant` via Groq for fast, intelligent responses |
| рҹҺҜ **Medical Focus** | Politely redirects off-topic queries back to health-related discussions |
| вҡЎ **Real-time Streaming UI** | Clean chat bubbles with a responsive Streamlit interface |
| рҹ”„ **Session Management** | Clear chat anytime via the sidebar |
| рҹ”’ **Secure Key Handling** | API keys loaded via `.env` locally or `st.secrets` on deployment |

## рҹҡҖ Getting Started

### Prerequisites

Python 3.10+ &nbsp;|&nbsp; A [Groq API key](https://console.groq.com/) *(free tier available)*

<br/>

### 1пёҸвғЈ Clone the Repository

```bash
git clone https://github.com/shreya/medibot.git
cd medibot
```

### 2пёҸвғЈ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3пёҸвғЈ Set Up Environment Variables

```env
GROQ_API_KEY=your_groq_api_key_here
```

### 4пёҸвғЈ Run the App

```bash
streamlit run app.py
```

Visit `http://localhost:8501` in your browser вҖ” MediBot is live! рҹҺү

## рҹҢҗ Deployment (Streamlit Cloud)

**1.** Push your code to a **public GitHub repository**<br/>
**2.** Go to [streamlit.io/cloud](https://streamlit.io/cloud) and connect your repo<br/>
**3.** In **App Settings вҶ’ Secrets**, add:

```toml
GROQ_API_KEY = "your_groq_api_key_here"
```

**4.** Deploy! Your app will be live at `https://your-app.streamlit.app` рҹҡҖ
